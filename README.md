# 🐱 Cat Photo App

A beginner-friendly HTML project that displays adorable cat photos using semantic HTML elements.  
This project was built as part of the [freeCodeCamp Responsive Web Design certification](https://www.freecodecamp.org/).

---

## 📸 Demo

🌐 [Live Demo](https://zahra-noorizadeh.github.io/cat-photo-app/)

---

## 🧰 Technologies Used

- HTML5
- Semantic Tags (`<figure>`, `<img>`, `<figcaption>`, etc.)
- Git & GitHub for version control and deployment

---

## 📁 Project Structure

cat-photo-app/
├── index.html
└── README.md


---

## ✨ Features

- Displays cat images using `<img>` with `alt` text
- Clean semantic structure using `<main>`, `<section>`, `<figure>`, and more
- Easy to extend with CSS or JavaScript later
